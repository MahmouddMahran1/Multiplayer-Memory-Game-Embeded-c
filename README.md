# Multiplayer-Memory-Game-Embeded-c
Multiplayer embedded memory game developed using Embedded C and ATmega328P microcontroller architecture.
An embedded systems project implementing a multiplayer memory game using Embedded C on the ATmega328P microcontroller.
The project features:
- Real-time gameplay
- State-machine architecture
- EEPROM score storage
- I2C LCD interfacing
- PWM audio generation
- Timer interrupt handling
- Multiple gameplay modes
- ## Features
- Multiplayer gameplay modes
- Real-time button input handling
- LCD menu interface
- EEPROM high-score saving
- Audio feedback system
- Difficulty scaling
- Idle sleep mode
- Modular multi-file architecture.
- ## Technologies Used
- Embedded C
- AVR Programming
- ATmega328P
- Arduino Uno
- I2C Communication
- Timer Interrupts
- EEPROM
- PWM
